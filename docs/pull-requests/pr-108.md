# [PR 108](https://github.com/vig-os/commit-action/pull/108) build(devkit): adopt devkit 1.4.0

## Summary

Re-scaffold this repo from **vigOS devkit 1.3.1 ‚Üí 1.4.0** (direnv mode) via the
installer (`--force --no-prompts --mode direnv`; values resolved from the
existing `.vig-os` manifest, mode/identity preserved).

This PR **pins `1.4.0-rc2` to validate it ahead of the final `1.4.0` release**;
the pins will be bumped to the final `1.4.0` revision before merge. Pins move in
lockstep:

| Pin | Value |
|-----|-------|
| `.vig-os` `DEVKIT_VERSION` | `1.4.0-rc2` |
| `flake.nix` `vigos.url` | `github:vig-os/devkit?ref=1.4.0-rc2` |
| `flake.lock` vigos rev | `6fe5f7704cc4d9d7894ff003ecdbd77746bf6ec7` (= tag `1.4.0-rc2^{}`) |

Preserved from `.vig-os`: `DEVKIT_MODE=direnv`, `DEVKIT_TAG_PREFIX=v`,
`DEVKIT_FLOATING_TAGS=major,minor`.
