# [PR 102](https://github.com/vig-os/commit-action/pull/102) ci(renovate): hold typescript major until toolchain supports v7

## Summary

Adds a Renovate `packageRules` entry that holds the `typescript` **major** update until our toolchain supports it.

TypeScript 7 (the native/Go compiler port) is excluded by our test/lint toolchain peers:
- `ts-jest@29` → `typescript >=4.3 <7`
- `typescript-eslint@8` → `typescript >=4.8.4 <6.1.0`

Renovate PR #99 (`typescript ^5.3.2 → ^7.0.0`) failed CI against these constraints (Tests + both lint jobs), plus an out-of-sync lockfile breaking `npm ci`. #99 is now closed.

Minor/patch `typescript` updates continue to flow; only the major is held. Remove this rule and bump `typescript` + `ts-jest` + `typescript-eslint` together once the peers ship TS 7-compatible releases.

Refs: #101
