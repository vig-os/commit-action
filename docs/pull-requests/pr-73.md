# [PR 73](https://github.com/vig-os/commit-action/pull/73) build(dist): untrack dist/src byproducts and gate the whole dist/ tree

Closes #70.

## What

`dist/` tracked **32 declaration byproducts** under `dist/src/**` alongside the one real artifact, `dist/index.js`. The `.gitignore` rule meant to exclude them — `dist/__tests__` — never matched their actual path, `dist/src/__tests__`.

Root cause is broader than the issue assumed: `tsconfig.json` sets `outDir: ./dist` with `declaration` + `declarationMap`, so **both `npm run build` (tsc) and `npm run bundle` (ncc)** emit them. ncc's own output lists `dist/src/commit.d.ts` among its assets — a plain `just bundle` is enough to regenerate all 17 of them.

## Changes

1. `git rm -r dist/src` — untrack the byproducts.
2. `.gitignore`: dead `dist/__tests__` rule → `dist/*` + `!dist/index.js`, so only the runtime artifact `action.yml` executes is tracked, and *anything* the build drops into `dist/` is ignored by default.
3. Both freshness gates (`dist-check.yml`, `release-extension.yml`) broadened from `dist/index.js` to the whole `dist/` tree — now possible because the location-dependent `.d.ts.map` files (which embed absolute `file://` paths, and which forced #59 to scope narrowly) are gone from the index.

The gates now test `git status --porcelain -- dist/` instead of `git diff --exit-code`. That is strictly stronger: `git diff` only sees *tracked* files, so build output that ncc starts emitting and nobody commits would slip through. `git status` catches both drift and uncommitted artifacts.

## Verification

Simulated CI in a fresh clone of this branch (`npm ci` → `npm run bundle`):

- **No false positive** — ncc regenerates all 17 byproducts under `dist/src/`, and the gate is clean: `git status --porcelain -- dist/` is empty. This is the case that matters, since the gate rebuilds before checking.
- **Still fails on drift** — appending to `src/retry.ts` and rebuilding yields ` M dist/index.js` → gate exits 1.
- **Catches the new case** — an artifact emitted but untracked is flagged, where the old `git diff` gate reported clean.

`just precommit` (full suite) green.
